class graphnet.data.utilities.parquet_to_sqlite.ParquetToSQLiteConverter(parquet_path, mc_truth_table='mc_truth', excluded_fields=None)[source]

Bases: LoggerMixin

Converts Parquet files to a SQLite database. Each event in the parquet file(s) are assigned a unique event id. By default, every field in the parquet file(s) are extracted. One can choose to exclude certain fields by using the argument exclude_fields.
